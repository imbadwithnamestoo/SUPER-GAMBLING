# 🎰 Super Gambling

Welcome to **Super Gambling**! This is a project I’ve been working on to practice C++. It started as a simple "guess the number" script, but it’s grown into a 800+ line terminal game with items, save files, and a full progression system.

The goal is pretty simple: Guess the right number, bet your points wisely, and try to collect every badge without going bankrupt.

### ✨ Game Features
*   **The Shop:** Buy items like **Extra Lives**, **Double Down**, and **Safe Bet Shields** to help you stay in the game.
*   **The Badge System:** 11 unique badges to unlock, including *High Roller*, *Streak Master*, and *Legend*.
*   **Extreme Mode:** For when you want a challenge. Every time you guess wrong, the secret number shifts by 3.
*   **Jackpots:** Every win has a random chance to trigger a Jackpot for a massive payout.
*   **Persistence:** Your stats, points, and badges are all saved to `save.txt` so you can keep playing later.
